<h1 align="center"> Smart Irrigation System</h1>

## **PROJECT SUMMARY:**

Smart irrigation is an Iot-based project which aims at making the job of irrigation a little easier. The project uses Raspberry Pi Pico and has been coded using Thonny and Micropython.
# Components Used:
* Raspberry Pi Pico WH
* Soil moisture Sensor
* Motor for supplying water
* Relay
* Jumper wires
* Power supply (9V battery)
* Blynk IoT Platform
* # Objectives:
* To irrigate small plants or even large fields in a smart way, without the need for human intervention.
* To make jobs less labour-absorbing.
# The Basic Working: 
When the moisture in the soil is low, the motor turns on to supply water. If the moisture level becomes sufficient the motor turns off.
# Advantages:
* Users can view live soil moisture readings and manually control the
water pump from a mobile application.
* This system is suitable for home gardens, terrace gardens, schools, and sma l farms, making it
a practical and real-world application of IoT technology.
